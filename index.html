Tables

Tag/Attribute

Name

Description
<table> Table Wrapper element for HTML tables
<thead> Table Head Wrapper element for the header row(s) of tables
<tbody> Table Body Wrapper element for body of a table
<tr> Table Row Creates a row for a table
<td> Table Data The data within the body of a table. Each <td> within a <tr> creates another column
<th> Table Header The data within the head of a table (under <thead>). Used in lieu of <td>
<tfoot> Table Foot Wrapper element for the footer row(s) of the table
colspan="x" Column Span Attribute of a table data or table head element to make it span multiple columns
rowspan="x" Row Span Attribute of a table data or table head element to make it span multiple rows

HTML Structure

Tag/Attribute

Name

Description
<a> Achor Used to create a link
href="x" Hyperlink reference Providing the address for the link
target="_blank" Target blank Opens link in new tab
<h1> - <h6> Headers 1-6 Size/importance of heading text
<p> Paragraph Standard paragraph element
<div> Divider Creates a section in which other elements can be contained
<!DOCTYPE html> Doctype declaration Tells browser the contained file is HTML
<html> HTML element Goes just under DOCTYPE declaration, tells browser that all contained code is HTML
<head> Head element Contains metadata of the page. Information not displayed on page
<body> Body element Contains displayed data of the page
<header> Header block element HTML5 specific element for designating header
<footer> Footer block element HTML5 specific element for designating footer
<nav> Navigational block element HTML5 specific element for designating a navigational block
<article> Article container element HTML5 specific element for any section that could stand alone
<aside> Aside element HTML5 specific element for supplementary information, whether in or out of an article element
<section> Section element HTML5 specific element for grouping related content together
<hgroup> Heading group HTML5 specific element for grouping multiple h1-h6 headings
<figure> Figure element HTML5 specific element for content referenced from the main flow of the article. Should contain a <figcaption>
<figcaption> Figure caption Used within <figure> to provide a text description of the figure
